Flow

Adding a widget (curated): Click widget type in catalog → panel switches to configuration form pre-filled with sensible defaults for that type → user adjusts title/measure if needed → "Add to Dashboard" places it at the bottom of the grid.

Adding a widget (advanced): Click "Advanced Configuration" toggle → full form appears with all data source, visualization, and filter options → user builds custom widget from scratch.

Editing existing widget: Click gear icon on widget → same panel opens pre-filled with current config → changes save on "Update Widget."
